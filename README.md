# LeetCode Solutions

My LeetCode solutions synced automatically.

## Project Structure

```text
leetcode-solutions/
│
├── .github/
│   └── workflows/      # GitHub Actions workflows for automated sync or testing
│
├── scripts/            # Script folder for automation tools (sync, test runner, template generator)
├── templates/          # Code templates for starting new solutions (e.g., Python, C++, Java)
│
├── config.json         # Configuration settings for sync settings and credentials
├── requirements.txt    # Python dependencies for setup and running scripts
└── README.md           # Documentation for the repository
```

## Getting Started

1. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Configure settings in [config.json](file:///d:/leetcode-solutions/config.json).
